# node 后台测试 demo

> 技术栈 nodejs koa2 koa-router typescript sequelize ...

试写后端接口

``` bash
# install dependencies
$ yarn install
# or
$ npm install

# 实施编译 ts
$ yarn run watch
# or
$ npm run watch

# serve with hot reload at localhost:3000
$ yarn run dev
($ yarn run serve)
# or
$ npm run dev
($ npm run serve)

# build for production
$ yarn run build
# or
$ npm run build
```
